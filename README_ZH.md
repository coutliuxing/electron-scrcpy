# electron-scrcpy
基于scrcpy的投屏软件，支持设备自动发现、导航栏、剪切板、拖拽安装、右键返回、控制远程设备

![screenshot](screenshot/screenshot.gif)
#### 如何安装

``` bash

npm install or yarn install

# 启动之后，会在9080端口监听
# 需要重新运行一次此命令
npm run dev

# build命令在不同系统环境中，需要的的不一样，需要自己根据自身环境进行配置
npm run build

```
